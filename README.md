# 🚀 KnowledgeVerse – Interactive Learning & Entertainment Hub

## 📌 Project Overview

**KnowledgeVerse – Interactive Learning & Entertainment Hub** is a responsive multi-page web application developed as part of **Task 3** of my **Web Development Internship**.

The project combines learning resources, entertainment, API integration, and an interactive quiz into a single platform. It provides users with a modern, engaging, and responsive experience while demonstrating essential front-end web development concepts.

---

## 🎯 Objectives

* Develop a responsive website using HTML, CSS, and JavaScript.
* Implement responsive design using CSS Media Queries.
* Integrate public APIs to fetch and display dynamic content.
* Create an interactive quiz application with a reward system.
* Build a professional multi-page web application with an attractive user interface.

---

## 🔄 Application Workflow

1. User opens the Login Page.
2. After logging in, the Dashboard is displayed.
3. Users can explore:

   * Weather
   * Quotes
   * Movies
   * Study Hub
   * Learning Videos
   * Quiz Challenge
4. The quiz calculates the user's score.
5. A reward page displays the achievement level and appreciation message.
